# Features（目指した課題解決）
ペルソナ
  
TryAppではジムや自主トレーニングをする人全てを対象にしています。
  
ユーザーストーリー
  
ジムに行っても三日坊主になってしまいがち、コミュニケーションしながらトレーニング、日々の記録をスマホで管理したい。というユーザーの課題を解決します。


# definition

- イントロダクション
    - スライド式の簡易的なアプリ説明ができる
    - Skipボタンがある
    - １度ログインした後は再度表示させる必要はない

- ログイン画面
    - ユーザー名を入力するフィールドがある
    - 身長を入力するフィールドがある
    - ラベルで簡易的な説明がある
    - 決定ボタンがある


- メイン画面（グラフ表示）
    - 体重の増減を数字で表示するラベルがある
    - 体重の増減を確認できるグラフがある
    - ランキング画面へ遷移させるボタンがある
    - 年月を選択できるチャートがある
    - （統一）タブバーからの画面遷移がある


- ランキング表示
    - ランキングだとわかるラベルがある
    - メニューボタンへ戻るボタンがある
    - ランキングを表示させる埋め込み型のviewがある


- 記録画面
    - 今日の日付を表示するラベルがある
    - 今日の体重を入力するフィールド、ラベルがある
    - 適正体重を計算するラベルがある
    - 記録するボタンがある


- 変更画面
    - 身長を入力するフィールドがある
    - 目標体重を入力するフィールドとラベルがある
    - 決定ボタンがある


- チャットルーム
    - チャット一覧を表示させることができる
    - チャット入力フィールドがある
    - 送信ボタンがある

- アウトプット画面１
    - アウトプットメニューとわかるラベルがある
    - その時のお題を表示させるテキストラベルがある
    - アウトプット可能な入力項目がある
    - みんなの投稿を確認するボタンがある
    - 送信ボタンがある
    
    
- アウトプット画面２
    - ナビゲーションバーは表示させて戻るボタンがある
    - お題を確認するラベルがある
    - 投稿一覧を確認できる表示をする
    - いいね機能がある
        


- アウトプット画面３
    - 各投稿にコメント機能がある
    - テキスト入力フィールドがある
    - 送信ボタンがある
    
    
   

- ログイン画面(２回目以降)
    - 記録するというラベルボタンがある
    - 埋め込み動画がある
